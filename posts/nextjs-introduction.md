# Next.js 入门指南

Next.js 是一个强大的 React 框架，用于构建现代化的 Web 应用。

## 核心特性

### 1. **服务端渲染（SSR）**

- 提升首屏加载速度
- 更好的 SEO 优化

### 2. **静态站点生成（SSG）**

- 构建时生成静态 HTML
- 极快的访问速度

### 3. **API 路由**

- 构建 API 端点
- 服务端逻辑处理

### 4. **文件系统路由**

- 基于文件的路由系统
- 简单直观

## 开始使用

使用 `npx create-next-app@latest` 创建新项目。

## 总结

Next.js 是构建现代 Web 应用的优秀选择。
